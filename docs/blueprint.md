# **App Name**: Volo Elevate

## Core Features:

- Product Showcase: Display Volo's lift and elevator series (Alto, Relux, Austre) with detailed specs, pricing, and features on dedicated product pages.
- Contact Form: Allow users to submit inquiries via a contact form, capturing their name, email, and message.
- Interactive Map: Embed a Google Maps instance to display the company's Bengaluru location, enhancing local discoverability.
- Social Media Integration: Link to company social media profiles in the footer, to expand the user's options to interact with the company.
- Mobile-Responsive Layout: Ensure a seamless experience across devices with a fully responsive design.

## Style Guidelines:

- Primary color: Dark Blue (#191970) to evoke luxury and elegance.
- Background color: Light silver (#F5F5F5) to create a clean and modern backdrop.
- Accent color: Grey (#808080) for key actions and text elements to ensure prominence.
- Body and headline font: 'Montserrat' (sans-serif) to maintain a contemporary and easily readable style. Note: currently only Google Fonts are supported.
- Employ soft shadows and generous rounded corners for a sophisticated and approachable visual style.
- Utilize Framer Motion for subtle fade-in and slide-up effects when transitioning between pages and sections.
- Incorporate minimal, clean icons that complement the luxury aesthetic of the site.
